# tic-tac-toe
Live Site: https://jakesummer.github.io/tic-tac-toe/

## Features
* Bot that plays the game against you
    * Toggle between human/bot by clicking the button on either side
    * Can also have 2 bots playing against each other!
* Change the color of your symbol by clicking the big "X" or "O" on either side
* Change each player's name by clicking their name on either side
* Score count to keep track of each player's score
